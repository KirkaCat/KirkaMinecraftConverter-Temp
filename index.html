Kirka x Minecraft Map Converter

Convert a Kirka map to a Minecraft schematic file or a schematic file to a Kirka map code.
Made for Minecraft 1.20

Note:
Everything was made and tested using 1.20.6. Use other 1.20 versions at your own risk, though it should be fine :)


Convert:

  • Kirka ➜ Minecraft
  • Minecraft ➜ Kirka

Mods:
  • Fabric Loader Installer
  • Fabric API (direct)
  • Sodium (direct) - A mod that optimizes Minecraft, drastically increasing performance.
  • WorldEdit (direct)

Resources:
  • Resource pack
  • Minecraft block pallette schematic
  • Kirka block pallete map code
